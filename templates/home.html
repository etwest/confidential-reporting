What Is This?

This is a demo site for demonstrating a method for making confidential reports online and for presenting resources to those who need them. This project is open source and is currently hosted on my github so feel free to copy that code if you have any need of it. The goal of this project was to be able to collect reports without having to rely upon a secure server or hosting service. Reports are collected and encrypted on the user's machine (see privacy) and passed through the served still encrypted.
